# AC测评工具 - 产品需求文档

## 1. 产品概述

**产品定位**：面向独立顾问的轻量级AC测评工具，采用"工具箱"模式，无需复杂项目管理，直接生成所需文档。

**核心价值**：将AC测评中最耗时的内容生成（胜任力模型、题本、报告）环节AI化，顾问聚焦专业判断。

**技术架构**：React前端 + Python/Node.js后端API

**数据策略**：本地优先（IndexedDB）+ 云端可选（免登录，云端备份需登录）

### 2.1 生成端旅程（流A）

```
步骤1: 胜任力模型
├── 上传文件（图片/Word/Excel/PDF，支持多文件）
├── 输入胜任力模型名称（多个，用逗号分隔）
├── 点击"AI生成"
├── AI解析输入，生成胜任力模型结构
├── 用户编辑确认（增删维度、调整行为等级）
└── 点击"保存并继续"

步骤2: 评估矩阵
├── 基于胜任力模型，AI生成评估矩阵
├── 用户可增删测评工具（BEI/LGD/角色扮演/案例分析/个人愿景）
├── 用户可调整各工具权重
├── 点击"保存并继续"

步骤3: 题本生成
├── 点击"生成题本"
├── AI根据评估矩阵，生成5个独立练习的题本
├── 在线预览（每个练习独立Tab）
├── 单个下载（每个练习可单独下载Word/PDF）
└── 批量下载（一键打包下载全部）

返回首页或继续其他流程
```

### 2.2 分析端旅程（流B）

```
步骤1: 上传测评记录
├── 下载固定模板（分数统计表 + 行为记录表）
├── 按模板填写被测者数据
├── 上传填写后的文件（Excel格式）
├── 系统解析并展示数据预览
├── 用户确认数据无误
└── 点击"保存记录"

步骤2: 生成反馈材料
├── 选择被测者（单个或批量）
├── 点击"生成反馈信"
├── AI根据测评记录生成个人反馈信
├── 在线预览
└── 下载Word文件

步骤3: 生成测评报告
├── 选择报告类型：组织版 / 个人版
├── 点击"生成报告"
├── AI根据测评记录生成报告
├── 在线预览
├── 下载Word/PDF
└── 批量下载（同时下载组织版+个人版）
```

**功能**：工具箱入口，展示5大功能模块

**页面元素**：

## 6. 设计风格

### 6.1 整体风格

**关键词**：简约、活力、专业

**视觉感受**：
- 界面干净，减少视觉噪音，让用户专注内容生成
- 适度使用动效和色彩，增加使用愉悦感
- 专业感与亲和力并存，适合顾问类用户

### 6.2 色彩系统

### 6.6 组件风格

**按钮**：
- 主按钮：蓝色填充，白色文字，圆角8px，hover加深
- 次按钮：白色填充，蓝色边框，蓝色文字
- 文字按钮：无边框无背景，蓝色文字

**卡片**：
- 白色背景，圆角12px，轻微阴影
- Hover时阴影加深，轻微上移

**输入框**：
- 白色背景，灰色边框，圆角8px
- Focus时蓝色边框，轻微阴影

**表格**：
- 斑马纹背景（奇偶行不同）
- 表头灰色背景加粗
- Hover行高亮

**模态框**：
- 白色背景，大圆角
- 背景遮罩半透明黑色

### 6.7 动效建议

**过渡时长**：
- 快速：150ms（按钮hover、输入框focus）
- 正常：250ms（卡片展开、Tab切换）
- 慢速：350ms（页面切换、模态框弹出）

**缓动函数**：
- 常规：`ease-out`
- 弹性：`cubic-bezier(0.34, 1.56, 0.64, 1)`

**动效场景**：
- 文件上传：进度条 + 完成时的打勾动画
- AI生成：骨架屏 + 打字机效果
- Tab切换：滑块跟随动画
- 下载：按钮状态变化 + 成功提示

## 7. 技术选型建议

### 前端
- **框架**：React 18 + TypeScript
- **UI库**：Tailwind CSS + shadcn/ui（可定制）
- **状态管理**：Zustand（轻量）

- **文档生成**：Python-docx + reportlab / Node-docx + puppeteer
- **文件存储**：本地文件系统 + 可选OSS

### 部署
- **前端**：Vercel / Netlify / 静态部署
- **后端**：Railway / Render / 自建
- **数据库**：可选PostgreSQL（云端）
